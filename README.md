# Angular.js进阶教程

本教程是读完Angular.js高级编程的笔记和自己的一些见解。（仅供参考）
本教程默认你已经有一定js基础和Angular.js经验，想要提升自己Angular编程的技巧，想要利用Angular.js来架构，以提升团队的开发和协作效率；避免走不必要的弯路。

每一个章节都可以单独学习，不一定按照顺序来学习。
